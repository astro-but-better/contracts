# @astro-turf/contracts

This package defines the semantic interfaces used by the Astro-TURF theming system.

Namespaces:
- Chrome     — site presentation data (navigation, authors, footer)
- Domain     — document semantics (Page, PageIndex, BlogPost, DocumentationPage…)
- Capability — functional semantics (search UI, related graph, outline…)

Includes:
- ThemeContract
- ThemeProviders
- buildTheme() DI helper
